## ğŸ”§ Custom Domain Setup

### DNS Configuration
Add these records to your domain registrar:

**For GitHub Pages:**
```
Type: CNAME
Name: www
Value: yourusername.github.io
```

**For Netlify/Vercel:**
```
Type: CNAME
Name: www
Value: your-site.netlify.app (or vercel.app)
```
